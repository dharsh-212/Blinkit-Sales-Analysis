# Blinkit-Sales-Analysis
Interactive Power BI dashboard analyzing Blinkit grocery sales, customer ratings, outlet performance, and product insights using business intelligence techniques.
Project Overview
Developed an interactive Power BI dashboard to analyze Blinkit grocery sales and business performance.
Tools Used
Power BI
Power Query
DAX
Excel

Total Sales
Average Sales
Number of Items
Average Rating
Dashboard Features
Sales by Item Type
Sales by Outlet Size
Sales by Outlet Location
Fat Content Analysis
Outlet Establishment Trend
Interactive Filters (Slicers)
Business Insights
Top-performing outlet types
Best-selling product categories
Sales distribution across locations
Customer rating analysis
